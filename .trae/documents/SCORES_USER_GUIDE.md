# 学生成绩查询功能使用指南

## 🎯 功能入口

### 访问路径
1. 登录学生账户
2. 在左侧导航栏点击 **"成绩查询"**
3. 或直接访问: `http://localhost:5173/dashboard/scores`

---

## 📊 页面功能说明

### 1. 统计卡片
页面顶部显示4个统计卡片：

| 卡片 | 说明 | 图标颜色 |
|------|------|---------|
| **参赛次数** | 您参加的竞赛总数 | 蓝色 🏆 |
| **获奖次数** | 获得奖项的竞赛数量 | 黄色 🥇 |
| **平均分** | 所有竞赛成绩的平均值 | 绿色 📈 |
| **最高分** | 最好的竞赛成绩 | 紫色 ⭐ |

### 2. 搜索和筛选
在操作栏可以：
- 🔍 **搜索框**: 输入竞赛名称或团队名称进行搜索
- 🏅 **筛选器**: 按获奖等级筛选（一等奖、二等奖、三等奖、优秀奖）

### 3. 导出和打印
- 📥 **导出成绩**: 将成绩导出为 CSV 文件，可用 Excel 打开
- 🖨️ **打印**: 直接打印成绩单

### 4. 成绩表格
表格显示所有成绩记录，包括：

| 列名 | 说明 |
|------|------|
| 序号 | 记录序号 |
| 竞赛名称 | 参加的竞赛名称 |
| 参赛团队 | 所在团队名称 |
| 成绩 | 竞赛得分（带颜色标识） |
| 排名 | 竞赛排名（前三名显示奖牌图标） |
| 获奖情况 | 获奖等级标签 |
| 状态 | 已发布/未发布 |
| 发布时间 | 成绩发布时间 |
| 操作 | 查看详情按钮 |

## 📖 详细操作指南

### 查看成绩详情
1. 在表格中找到要查看的成绩记录
2. 点击操作列的 **"详情"** 按钮
3. 弹出对话框显示完整成绩信息
4. 可在对话框中点击 **"打印成绩单"** 打印单条成绩

### 导出成绩
1. 使用搜索或筛选功能找到需要导出的成绩
2. 点击右上角的 **"导出成绩"** 按钮
3. 系统自动下载 CSV 文件
4. 文件名格式: `我的成绩_YYYY-MM-DD.csv`
5. 可使用 Excel、Numbers 等软件打开

### 打印成绩单
#### 打印所有成绩
1. 点击右上角的 **"打印"** 按钮
2. 浏览器打开打印预览
3. 选择打印机或保存为 PDF
4. 完成打印

#### 打印单条成绩
1. 点击成绩记录的 **"详情"** 按钮
2. 在弹出对话框中点击 **"打印成绩单"**
3. 浏览器打开打印预览
4. 完成打印

### 搜索成绩
1. 在搜索框中输入关键词
   - 可搜索竞赛名称（如"数学建模"）
   - 可搜索团队名称（如"算法小组"）
2. 实时显示搜索结果
3. 点击搜索框右侧的 ❌ 清除搜索

### 筛选成绩
1. 点击 **"筛选获奖等级"** 下拉框
2. 选择要查看的获奖等级
   - 全部
   - 一等奖
   - 二等奖
   - 三等奖
   - 优秀奖
3. 表格显示筛选后的结果
4. 点击下拉框右侧的 ❌ 清除筛选

### 排序成绩
- 点击 **"成绩"** 列头可按分数排序
- 点击 **"排名"** 列头可按名次排序
- 再次点击切换升序/降序

## 🚫 常见问题

### Q1: 为什么看不到成绩？
**A**: 可能的原因：
- 您还没有参加任何竞赛
- 竞赛成绩尚未发布
- 您的团队还没有提交答卷

**解决方法**:
- 点击页面中的 **"浏览竞赛"** 按钮报名参赛
- 等待教师发布成绩

### Q2: 为什么有些成绩显示"未发布"？
**A**: 教师已经评完分但还没有正式发布成绩。
- 未发布状态：成绩仅教师可见
- 已发布状态：学生可以查看成绩

### Q3: 可以修改已发布的成绩吗？
**A**: 不可以。学生只能查看成绩，不能修改。
- 如发现成绩有误，请联系授课教师
- 教师可以重新评分和发布

### Q4: 导出的 CSV 文件乱码怎么办？
**A**: 使用 Excel 打开 CSV 文件的正确方法：
1. 打开 Excel
2. 点击 **"数据"** → **"从文本/CSV"**
3. 选择导出的文件
4. 文件编码选择 **"UTF-8"**
5. 点击 **"加载"**

### Q5: 为什么没有排名？
**A**: 部分竞赛可能不设排名，或教师尚未设置排名。

### Q6: 证书在哪里下载？
**A**: 如果竞赛提供证书，在成绩详情中会显示 **"下载证书"** 按钮。
- 目前该功能正在开发中
- 请关注后续更新

## 💡 使用技巧

### 技巧 1: 快速查找特定竞赛成绩
1. 使用搜索框输入竞赛名称的部分文字
2. 快速定位到目标成绩

### 技巧 2: 导出优秀成绩
1. 使用筛选器选择获奖等级
2. 点击导出按钮
3. 只导出获奖的成绩记录

### 技巧 3: 打印精美成绩单
1. 打开打印对话框
2. 选择 **"保存为 PDF"**
3. 保存后可分享给家长或用于求职简历

### 技巧 4: 对比历史成绩
1. 按成绩列排序
2. 查看成绩进步情况
3. 总结学习经验

## 🎓 学习建议

### 利用成绩数据
1. **定期查看**: 及时了解竞赛表现

3. **奖学金申请**: 竞赛成绩可作为申请材料
4. **保研推荐**: 优秀成绩有助于保研
